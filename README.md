# Nguyễn Minh Trung - Personal Blog

Static GitHub Pages portfolio/blog for cyber security notes, projects, labs, and writeups.

## Files

- `index.html`: homepage content and page structure.
- `assets/css/style.css`: all visual styling and responsive layout rules.
- `assets/img/avatar.jpg`: profile photo used on the left column.

## How To Update

1. Replace `assets/img/avatar.jpg` with another portrait photo if needed. Keep the same filename to avoid editing HTML.
2. Edit `index.html` to update your LinkedIn URL, email address, project descriptions, and writeup links.
3. Push the repo to GitHub and enable GitHub Pages for the repository.

GitHub Pages URL:

```text
https://trung-chigusa.github.io/
```
